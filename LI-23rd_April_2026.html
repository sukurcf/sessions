Slash Commands

Type / in Chat to trigger quick, focused actions — no need to type out full prompts.

Command What It Does Example
/explain Explains selected code Select a regex → /explain
/fix Proposes a fix for errors Select buggy code → /fix
/tests Generates unit tests Select a function → /tests
/doc Adds documentation comments Select a class → /doc
/new Scaffolds a new project/file /new Express API with TypeScript
💡 Pro Tip: Combine slash commands with context variables for better results — e.g., /explain #selection
🎯

Context Variables

Use # references to give Copilot specific context — the more context, the better the answer.

#file

Reference a specific file from your workspace for Copilot to analyze

#selection

Pass your currently selected code as context to the chat

#editor

Include the entire visible content of your active editor

#codebase

Let Copilot search your entire project for relevant context

#terminalLastCommand

Share the last terminal command and its output with Copilot

#problems

Include current editor errors and warnings for debugging help

📌 Usage: Type # in the chat input to see all available context variables — then pick the right one.

Chat Participants

Type @ to summon specialized AI agents — each expert in a different domain.

@workspace

Understands your entire project — ask about files, structure, dependencies, and how things connect.

Find files Project Q&A Architecture

@vscode

Expert on VS Code itself — settings, keybindings, extensions, and editor configuration.

Settings Keybindings Extensions

@terminal

Helps with shell commands, scripts, and terminal operations — suggests the right CLI commands.

CLI Help Scripts Shell

@github

Searches GitHub for repos, issues, PRs, and code — great for finding examples and references.

Search Issues Repos
