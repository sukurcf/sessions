Section 2 · Coding & Review

Coding Agent vs. Agent Mode

Two ways to leverage Copilot's autonomous capabilities

🔧 Coding Agent

☁️ Runs in GitHub Actions (cloud)
🔀 Opens PRs autonomously
📝 Triggered from Issues / PR comments
👥 Team-visible — full audit trail
⏳ Async — works in background
🔒 Scoped to one repo per task
VS

🖥️ Agent Mode

💻 Runs in your local IDE
✏️ Edits files directly in editor
💬 Triggered from Chat in VS Code
👤 Personal — local changes only
⚡ Sync — real-time interaction
🔓 Access to entire workspace

Section 2 · Coding & Review

Copilot Code Review GA

AI-powered code review with inline suggestions and one-click fixes

🔀 Open PR
↓
🤖 Copilot reviews changes
↓
💬 Inline comments + suggestions
↓
✅ One-click fix or batch commit

How to request

  • Select "Copilot" from Reviewers menu on PR
  • Configure automatic reviews via repo rules
  • Apply fixes with one click or batch commit
  • Invoke coding agent for complex fixes

📌 Note: Copilot leaves "Comment" reviews — it never "Approves" or "Requests changes" and won't block merging.
